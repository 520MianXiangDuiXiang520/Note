# 51单片机

## 单片机是什么

1. 一种集成电路芯片（可编程）
2. 采用超大规模集成电路计数
3. 计算机部件集成
4. 一个小而完善的微型计算机系统
5. 在工业控制领域广泛应用

特点：

1. 可靠性高
2. 便于拓展
3. 控制功能强，具有丰富的控制指令
4. 低电压，低功耗
5. 片内存储容量较小
6. 集成度高，体积小
7. 性价比高
8. 应用广泛
9. 易于产品化

功能：

* 主要用于控制（或受控于）外设。往往嵌入某个仪器/设备/系统中，使其达到智能化的效果。

应用领域：

* 民用电子产品、计算机系统中的部件控制器、智能仪器仪表、工业测控、网络与通信的智能接口、军工领域、办公自动化、集散控制系统、并行多机处理系统和局域网络系统。
