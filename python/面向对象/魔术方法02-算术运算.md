# 魔术方法02-算术运算

python一切皆对象

```python
>>> class A():
	pass

>>> type(A)
<class 'type'>
>>> type(int)
<class 'type'>
>>> type(type)
<class 'type'>
>>> type(dir)
<class 'builtin_function_or_method'>
>>> 
```
