# spring 概述

## 1. 什么是spring

1. 轻量的全栈开源框架
2. 以IoC（反转控制）和AOP（面向切面编程）为内核
3. 提供展现层spring MVC 和持久层Spring JDBC 以及业务层事务管理等众多应用技术
4. 整合了众多开源框架和类库
5. 最著名的和使用最多的企业级应用开源框架

## Spring的优势

1. 方便解耦，简便开发
2. 支持AOP编程
3. 支持申明式事务
4. 方便测试
5. 方便继承其他优秀框架
6. 降低JavaEE API使用难度
7. 源码经典
